# WMS 仓库管理系统 — 项目总览

## 这是什么项目

WMS (Warehouse Management System) 是一个**仓库管理系统**，用于管理商品、仓库、库位、入库、出库和库存查询。

技术栈：**Rust 后端**（Axum + SQLx + PostgreSQL）+ **React 前端**（Vite + Ant Design + TanStack Query）。

## 核心功能（MVP）

| 模块 | 功能 |
|------|------|
| 商品管理 | 新增、编辑、查询、启用/禁用商品 |
| 仓库管理 | 新增、编辑、查询仓库 |
| 库位管理 | 按仓库管理库位，支持多种库位类型 |
| 入库管理 | 创建入库单 → 填写实收数量 → 完成入库 → 库存自动增加 |
| 出库管理 | 创建出库单 → 填写实发数量 → 完成出库 → 库存自动扣减 |
| 库存查询 | 按商品/仓库/库位查询当前库存数量 |
| 库存流水 | 后台自动记录每次库存变更（增减前后数量），供审计使用 |

## 设计原则

- **分层架构**：Handler → Service → Repository → 数据库，每层职责明确
- **事务一致性**：所有涉及库存变更的操作都在数据库事务中完成
- **并发安全**：使用 CAS（Compare-And-Swap）防止订单状态并发冲突
- **无分类设计**：商品不设分类，简化 MVP 设计
- **无审批流**：入库/出库直接从 draft 到 completed，不设审批环节
